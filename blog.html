Question-1: Difference Between Local Storage ans Session Storage.

Answer:

NO. Local Storage Session Storage
1. Local Storage is not session-based, it must be deleted via javascript or manually. Session Storage is session-based and works per window or tab.
2 The storage capacity of local storage is 5MB/10MB. The storage capacity of local storage is 5MB.
3 Local Storage is never expire. That means until the data is not deleted it remains by default. Session Storage is expire on tab close.
